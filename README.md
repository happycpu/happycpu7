# 정시·논술 입시요강 웹사이트 (서울·수도권 상위 20교)

재수생·정시 준비생을 위한 입시요강 정리 사이트. **논술전형**과 **수능 정시**에 집중하고,
**수능 백분위를 입력하면 지원가능 대학·학과를 찾아주는** 기능을 제공합니다.
로그인 없음. 매일 1회 백그라운드로 입학처 페이지 변경을 감지하고, "다시 조사하기"로 즉시 재점검합니다.

## 대상 대학 (20교)
서울대·연세대·고려대를 제외한 서울·수도권 4년제 중 인지도 상위 20교
(성균관·한양·서강·중앙·경희·한국외대·서울시립·건국·동국·홍익·이화·숙명·국민·숭실·세종·광운·아주·인하·단국·가천).
목록/순위는 `src/universities.base.js` 에서 조정할 수 있습니다.

## 실행
```bash
npm install
npm start                 # http://localhost:3000
```
환경변수: `PORT`(기본 3000), `DAILY_CRON`(기본 "10 4 * * *"), `TZ`(기본 Asia/Seoul)

## 주요 기능
- **성적으로 찾기**: 국어·수학·탐구 백분위 입력 → 정시 입결 기반 안정/적정/소신/도전 판정 (`POST /api/match`)
- **대학 목록**: 논술·정시 요강 요약 카드, 상세 모달(전형방법·수능최저·정시 반영·입결 표·출처)
- **변경 이력**: 입학처 페이지 변경 감지 로그
- **자동 점검**: 매일 지정 시각에 전체 입학처 페이지를 크롤링해 콘텐츠 해시로 변경 감지 (`src/scheduler.js`)
- **다시 조사하기**: 헤더 버튼 → `POST /api/recheck` 즉시 재점검

## 데이터 갱신(입시전형·입결 리서치)
입시전형/입결 상세 데이터(`data/admissions.json`)는 대학별 AI 리서치 + 입학처 URL 실접속 검증
워크플로우로 수집합니다. 현재 스냅샷: **20개교 전원, 정시 입결 614건, 입학처 URL 20개 검증 완료**
(기준 2027학년도 전형계획 / 직전 정시 입결).

- 리서치 워크플로우 정의: `research/research-workflow.js`
  (Claude Code의 Workflow 도구로 실행하는 오케스트레이션 스크립트 — 대학 1곳당 전형·입결 조사 2개
  에이전트 병렬 + 검증 1개. `node`로 직접 실행하는 스크립트가 아님.)
- 워크플로우 결과 원본 스냅샷: `data/research-latest.json`
- 결과(JSON)를 `data/admissions.json`으로 반영:
  ```bash
  node src/importResearch.js data/research-latest.json
  ```
서버는 매 요청마다 `data/admissions.json`을 새로 읽으므로 데이터 갱신 후 재시작이 필요 없습니다.
입학처 URL이 갱신되면 사이트의 **다시 조사하기** 버튼으로 크롤러 baseline을 재설정하세요.

## 데이터 구조
- `src/universities.base.js` — 대학 기준 목록(순위·지역·후보 입학처 URL)
- `data/admissions.json` — 리서치로 수집된 논술·정시·입결 상세
- `data/state.json` — 크롤링 상태(마지막 확인시각·콘텐츠 해시·변경 이력)

## API
| 메서드 | 경로 | 설명 |
|---|---|---|
| GET | `/api/universities` | 대학 목록(전형·입결·점검상태 병합) |
| GET | `/api/universities/:slug` | 대학 상세 |
| GET | `/api/status` | 점검 상태·마지막 확인 시각 |
| GET | `/api/changelog` | 변경 이력 |
| POST | `/api/recheck` | 즉시 재점검(다시 조사하기) |
| POST | `/api/match` | 수능 백분위 → 지원가능 대학·학과 |

## 주의
입결·전형 정보는 공개자료 기반 **추정·정리치**로 오류가 있을 수 있습니다.
실제 지원 전 각 대학 **최신 모집요강**과 입시자료로 반드시 재확인하세요.
